## ⭐ Featured Project — [Sober](https://github.com/scitech-valley/sober-)

A **full-stack recovery companion platform** I built end-to-end — mobile app, admin panel, and API.

| Layer | Stack |
|-------|-------|
| **Mobile** | React Native · Expo · TypeScript |
| **Web Admin** | React · Vite · TypeScript |
| **Backend** | Node.js · Express · Prisma · MySQL |

**What it does (client-visible features):**

- 📊 **Sobriety tracking** — streak counter, milestones, goal progress ring
- ✅ **Daily check-ins** — mood, craving level, notes with streak history
- 📓 **Private journal** — create, edit, and reflect with image support
- 🌍 **Community stories** — share recovery journeys, likes & comments
- 📈 **My Journey dashboard** — consistency calendar, milestone timeline
- 🎯 **Personalized home feed** — featured stories & daily motivation
- 🔐 **Auth & profiles** — registration, social sign-in, avatar uploads
- 🛡️ **Admin panel** — user management, recovery data, content moderation

*This is the kind of full product delivery I bring to client projects — not just a single screen, but a complete, connected system.*
